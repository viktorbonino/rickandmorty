# Rick and Morty API Frontend
This is a simple responsive frontend for Rick and Morty API.

Live version: https://rickandmorty-lyart.vercel.app/

**Technologies/Libraries used**
- React
- Typescript
- TailwindCSS
- Axios
- React query
- Hero icons
- Vite
- Local Storage
- Custom hooks
